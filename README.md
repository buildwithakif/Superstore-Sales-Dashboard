# Super Store Sales Dashboard | Power BI Project

## Overview
This project is an interactive Power BI dashboard built using the Super Store Sales dataset. The dashboard provides valuable business insights related to sales performance, profit trends, customer segments, and regional analysis.

## Features
- Sales and Profit Analysis
- Regional Performance Tracking
- Category & Sub-Category Insights
- Customer Segment Analysis
- Monthly Sales Trends
- Interactive Filters and Slicers

## Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX
- CSV Dataset

## Dataset
The project uses the Super Store Sales dataset for data visualization and business analysis.

## Project Files
- `Super Store Sales Dashboard.pbix` → Main Power BI dashboard
- `SuperStore_Sales_Dataset.csv` → Dataset used in the project

## Dashboard Insights
The dashboard helps analyze:
- Total Sales
- Total Profit
- Sales by Region
- Top Performing Categories
- Customer Segment Contribution
- Monthly Growth Trends

## How to Use
1. Download the repository files.
2. Open the `.pbix` file in Power BI Desktop.
3. Refresh the dataset if needed.
4. Explore the interactive dashboard visuals.
